Research

My primary research interests are computational chemistry, graph neural networks, and immunology.

Under the mentorship of Forest Agostinelli at the AI Institute at USC, I've worked on a project applying deep reinforcement learning to chemistry. I've enjoyed expanding my machine learning skills in PyTorch, learning to work with a remote compute cluster, and producing informative visualizations.

As my undergraduate thesis, I conducted a literature review on Interleukin-12 in solid tumor immunotherapy with the help of my advisors, Dr. Katie Kathrein and Dr. Hexin Chen. I also analyzed data from the Cancer Genome Atlas on immune cell proportions in solid tumors. If you like reading abbreviations for long science terms, you can view the manuscript here.

Projects

Classifying Immune Cell Images with ResNet-18

I trained an image classification model with a pre-trained ResNet-18 backbone on microscopic immune cell images. After 47 epochs, my model achieved a test accuracy of 99.1% (Breaking Bad anyone?). You can view the code here. Also, I built a small command line interface where you can classify images locally, which you can access at this repo.

Logistic Regression From Scratch

I implemented logistic regression in pure numpy. My implementation contains 4 algorithms as solvers including batch gradient descent, mini-batch gradient descent, stochastic gradient descent, and stochastic average gradient. I also derived numerically stable versions of sigmoid and cross entropy. Finally, I applied my model to 4 toy datasets as well as used it to predict whether a patient has diabetes based on a set of known features. You can find my code as well as an explanation of what I learned here.

Classifying Medical Abstracts Using Different NLP Techniques

I attempted to use different supervised learning techniques to reproduce results from this paper to classify medical abstracts into one of five topics. The best accuracy was achieved using a fine-tuned clinical BERT model. Not my best models, but hey, I learned how (and how not) to do NLP stuff...

BoW = Bag of Words
SVC = Support Vector Classification
LR = (Multiclass) Logistic Regression
RF = Random Forest
Word2Vec = averaged pre-trained word embeddings
ScispaCy = word embeddings pre-trained on biomedical text
BERT = Bidirectional Encoder Representations from Transformers (an LLM)

Modeling Potency of CYP2D6 Inhibitors

I used a balanced random forest to model inhibition coefficient values for compounds targeting the CYP2D6 enzyme (data obtained from ChEMBL). The model achieved 75.42 % accuracy when trained on molecular descriptors. You can check out an informative powerpoint here and the associated code here.

CMS Healthcare Consulting Payments (2016-2022)

I aggregated, cleaned, categorized, and analyzed all available data on payments made to healthcare consultants from 2016 to 2022 from the Center for Medicare and Medicaid Services (CMS). You can view my analysis organized into a tableau dashboard here.
